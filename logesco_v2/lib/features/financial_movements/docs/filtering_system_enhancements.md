# Système de Filtrage Amélioré - Mouvements Financiers

## Vue d'ensemble

Le système de filtrage des mouvements financiers a été considérablement amélioré avec de nouvelles fonctionnalités avancées, une meilleure expérience utilisateur et des optimisations de performance.

## Nouvelles Fonctionnalités

### 1. Recherche Avancée Multi-Champs
- **Recherche par description** : Recherche dans les descriptions des mouvements
- **Recherche par référence** : Recherche par numéro de référence
- **Recherche par notes** : Recherche dans les notes associées
- **Recherche par utilisateur** : Recherche par nom d'utilisateur
- **Syntaxe avancée** : Support des préfixes (desc:, ref:, notes:, user:)

### 2. Presets de Filtres Sauvegardés
- **Presets par défaut** : Filtres rapides prédéfinis (Aujourd'hui, Cette semaine, Ce mois, Montants élevés)
- **Presets personnalisés** : Sauvegarde des combinaisons de filtres personnalisées
- **Gestion des presets** : Interface pour créer, modifier et supprimer les presets
- **Application rapide** : Application en un clic des presets sauvegardés
