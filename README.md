# Logesco V2

Système de gestion commerciale complet avec interface Flutter et backend Node.js/Express.

## 🚀 Fonctionnalités principales

- Gestion des ventes et caisse
- Gestion des stocks et inventaire
- Gestion des clients et fournisseurs
- Comptabilité et bilan comptable
- Système de permissions granulaires
- Gestion des dates de péremption
- Import/Export Excel
- Impression thermique
- Système de licences

## 📋 Prérequis

- Node.js 18+
- Flutter 3.x
- PostgreSQL ou SQLite
- Windows (pour le build desktop)

## 🛠️ Installation

### Backend
```bash
cd backend
npm install
npm run setup
npm start
```

### Client Flutter
```bash
cd logesco_v2
flutter pub get
flutter run -d windows
```
