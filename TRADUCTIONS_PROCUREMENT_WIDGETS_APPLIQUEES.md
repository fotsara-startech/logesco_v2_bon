# Traductions appliquées au module Procurement - Widgets

## Résumé des modifications

Les traductions ont été ajoutées aux fichiers de traduction français (`fr_translations.dart`) et doivent maintenant être appliquées aux widgets du module procurement.

## Clés de traduction ajoutées

### Français (fr_translations.dart) ✅
- Dialog création commande (create_commande_dialog.dart)
- Dialog sélection produit
- Détails commande (commande_details_dialog.dart)
- Card commande (commande_card.dart)
- Tooltips et labels

### Anglais (en_translations.dart) - À ajouter

Les mêmes clés doivent être ajoutées en anglais dans `en_translations.dart`.

1. **create_commande_dialog.dart** - Dialog de création de commande
2. **commande_card.dart** - Carte d'affichage de commande
3. **commande_details_dialog.dart** - Dialog de détails de commande
4. **receive_commande_dialog.dart** - Dialog de réception
5. **cancel_commande_dialog.dart** - Dialog d'annulation

7. **alertes_approvisionnement_widget.dart** - Widget d'alertes
