## Vidéo 1: Introduction et Première Connexion

**Durée** : 5-7 minutes  
**Objectif** : Présenter LOGESCO v2 et guider la première connexion

### Script

**[00:00 - 00:30] Introduction**

> "Bonjour et bienvenue dans cette série de tutoriels sur LOGESCO v2, votre nouveau logiciel de gestion commerciale. Je suis [Nom], et je vais vous guider pas à pas dans l'utilisation de LOGESCO."

**[00:30 - 01:00] Présentation de LOGESCO**

> "LOGESCO v2 est un logiciel complet qui vous permet de gérer tous les aspects de votre commerce : produits, clients, ventes, stock, et bien plus encore. Il fonctionne entièrement sur votre ordinateur, sans besoin d'internet."

**[01:00 - 02:00] Démarrage de l'Application**

> "Pour démarrer LOGESCO, double-cliquez simplement sur l'icône sur votre bureau."
> 
> [ACTION: Double-cliquer sur l'icône]
> 
> "L'application prend quelques secondes à démarrer. Vous verrez d'abord un écran de chargement, puis l'écran de connexion apparaîtra."

**[02:00 - 03:30] Première Connexion**

> "Pour votre première connexion, utilisez les identifiants par défaut :"
> - "Nom d'utilisateur : admin"
> - "Mot de passe : admin123"
> 
> [ACTION: Taper les identifiants]
> 
> "Cliquez maintenant sur Se connecter."
> 
> [ACTION: Cliquer sur Se connecter]
> 
> "Important : après votre première connexion, je vous recommande fortement de changer ce mot de passe pour sécuriser votre système."

**[03:30 - 05:00] Découverte de l'Interface**

> "Vous voici maintenant sur le tableau de bord de LOGESCO. Prenons un moment pour découvrir l'interface."
> 
> "Sur la gauche, vous avez le menu principal avec tous les modules :"
> - "Tableau de Bord : pour une vue d'ensemble"
> - "Produits : pour gérer votre catalogue"
> - "Clients et Fournisseurs : pour vos contacts"
> - "Stock : pour suivre vos quantités"
> - "Ventes : pour enregistrer vos transactions"
> 
> [ACTION: Survoler chaque élément du menu]

**[05:00 - 06:30] Changer le Mot de Passe**

> "Maintenant, changeons ce mot de passe par défaut. Cliquez sur votre nom en haut à droite."
> 
> [ACTION: Cliquer sur le nom d'utilisateur]
> 
> "Sélectionnez Paramètres, puis Changer le mot de passe."
> 
> [ACTION: Naviguer vers le changement de mot de passe]
> 
> "Entrez votre ancien mot de passe, puis votre nouveau mot de passe deux fois. Choisissez un mot de passe fort avec au moins 8 caractères."
> 
> [ACTION: Remplir le formulaire]
> 
> "Cliquez sur Enregistrer. Votre mot de passe est maintenant sécurisé."

**[06:30 - 07:00] Conclusion**

> "Félicitations ! Vous êtes maintenant connecté à LOGESCO et prêt à commencer. Dans la prochaine vidéo, nous verrons comment ajouter vos premiers produits au catalogue."
> 
> "N'hésitez pas à explorer l'interface par vous-même. À bientôt !"

### Points Clés à Montrer

- ✅ Icône de l'application sur le bureau
- ✅ Écran de chargement
- ✅ Formulaire de connexion
- ✅ Interface principale
- ✅ Menu latéral
- ✅ Changement de mot de passe

### Matériel Nécessaire

- Enregistrement d'écran en 1920x1080
- Microphone de qualité
- Musique d'intro/outro (optionnel)
- Annotations et flèches pour guider l'œil

## Vidéo 2: Gestion des Produits

**Durée** : 8-10 minutes  
**Objectif** : Apprendre à créer, modifier et gérer les produits

### Script

**[00:00 - 00:30] Introduction**

> "Bienvenue dans ce deuxième tutoriel LOGESCO. Aujourd'hui, nous allons apprendre à gérer votre catalogue de produits. C'est la base de votre système de gestion commerciale."

**[00:30 - 01:30] Accéder au Module Produits**

> "Pour commencer, cliquez sur Produits dans le menu de gauche."
> 
> [ACTION: Cliquer sur Produits]
> 
> "Vous voyez maintenant la liste de vos produits. Si c'est votre première utilisation, cette liste est vide. Nous allons y remédier en ajoutant notre premier produit."

**[01:30 - 05:00] Créer un Produit**

> "Cliquez sur le bouton + Nouveau Produit en haut à droite."
> 
> [ACTION: Cliquer sur + Nouveau Produit]
> 
> "Un formulaire s'ouvre. Remplissons-le ensemble :"
> 
> "Référence : C'est le code unique de votre produit. Par exemple, LAPTOP001."
> [ACTION: Taper LAPTOP001]
> 
> "Nom : Le nom commercial du produit. Par exemple, Ordinateur Portable Dell."
> [ACTION: Taper le nom]
> 
> "Prix Unitaire : Le prix de vente. Par exemple, 50000."
> [ACTION: Taper 50000]
> 
> "Description : Vous pouvez ajouter des détails. Par exemple, Ordinateur portable 15 pouces, 8GB RAM, 256GB SSD."
> [ACTION: Taper la description]
> 
> "Catégorie : Pour organiser vos produits. Par exemple, Électronique."
> [ACTION: Taper Électronique]
> 
> "Seuil Stock Minimum : La quantité en dessous de laquelle vous voulez être alerté. Par exemple, 5."
> [ACTION: Taper 5]
> 
> "Maintenant, cliquez sur Enregistrer."
> [ACTION: Cliquer sur Enregistrer]
> 
> "Parfait ! Votre premier produit est créé et apparaît dans la liste."
