## Introduction à la Formation

### Objectifs de la Formation

À la fin de cette formation, vous serez capable de :
- ✅ Naviguer efficacement dans LOGESCO v2
- ✅ Gérer le catalogue de produits
- ✅ Enregistrer et suivre les ventes
- ✅ Gérer le stock et les approvisionnements
- ✅ Suivre les comptes clients et fournisseurs
- ✅ Générer et interpréter les rapports

### Durée Estimée

- **Formation complète** : 4-6 heures
- **Formation express** : 2-3 heures (modules essentiels)
- **Rafraîchissement** : 1 heure

### Prérequis

- Connaissance de base de Windows
- Compréhension des concepts commerciaux de base
- Aucune expérience informatique avancée requise

## Module 1: Prise en Main

### Durée: 30 minutes

### 1.1 Démarrage de l'Application

**Objectif** : Apprendre à lancer et se connecter à LOGESCO v2

**Étapes** :
1. Double-cliquez sur l'icône LOGESCO sur le bureau
2. Attendez le chargement de l'application (10-15 secondes)
3. L'écran de connexion s'affiche

**[CAPTURE D'ÉCRAN: Écran de connexion]**

**Exercice Pratique** :
- Lancez l'application
- Observez l'écran de connexion

### 1.2 Connexion

**Identifiants par Défaut** :
- Nom d'utilisateur : `admin`
- Mot de passe : `admin123`

**Étapes** :
1. Entrez le nom d'utilisateur dans le premier champ
2. Entrez le mot de passe dans le second champ
3. Cliquez sur le bouton **Se connecter**

⚠️ **Important** : Changez le mot de passe après la première connexion !

**[CAPTURE D'ÉCRAN: Formulaire de connexion rempli]**

**Exercice Pratique** :
- Connectez-vous avec les identifiants par défaut
- Notez le temps de chargement

### 1.3 Découverte de l'Interface

**Objectif** : Se familiariser avec la disposition de l'interface

**Éléments Principaux** :

1. **Barre Supérieure**
   - Logo LOGESCO
   - Nom de l'utilisateur connecté
   - Bouton de déconnexion

2. **Menu Latéral Gauche**
   - Tableau de Bord
   - Produits
   - Clients
   - Fournisseurs
   - Stock
   - Approvisionnements
   - Ventes
   - Paramètres

3. **Zone Centrale**
   - Contenu principal
   - Formulaires et listes
   - Graphiques et statistiques

4. **Barre d'Actions**
   - Boutons d'action contextuels
   - Recherche
   - Filtres

**[CAPTURE D'ÉCRAN: Interface principale annotée]**

**Exercice Pratique** :
- Cliquez sur chaque élément du menu
- Observez le changement de contenu
- Revenez au tableau de bord

### 1.4 Navigation de Base

**Raccourcis Clavier** :
- `Ctrl + N` : Nouveau (produit, client, vente selon le contexte)
- `Ctrl + S` : Enregistrer
- `Ctrl + F` : Rechercher
- `Échap` : Fermer la fenêtre/dialogue
- `F5` : Actualiser les données

**Exercice Pratique** :
- Testez chaque raccourci clavier
- Naviguez entre les différents modules
- Pratiquez la recherche

**Quiz Module 1** :
1. Quels sont les identifiants par défaut ?
2. Où se trouve le bouton de déconnexion ?
3. Quel raccourci permet de créer un nouveau élément ?
4. Combien de modules principaux y a-t-il dans le menu ?

## Module 2: Gestion des Produits

### Durée: 45 minutes

### 2.1 Accéder au Module Produits

**Étapes** :
1. Cliquez sur **Produits** dans le menu latéral
2. La liste des produits s'affiche (vide au départ)

**[CAPTURE D'ÉCRAN: Liste des produits vide]**

### 2.2 Créer un Nouveau Produit

**Objectif** : Ajouter un produit au catalogue

**Étapes Détaillées** :

1. **Ouvrir le Formulaire**
   - Cliquez sur le bouton **+ Nouveau Produit**
   - Le formulaire de création s'affiche

2. **Remplir les Informations Obligatoires**
   - **Référence** : Code unique (ex: PROD001)
   - **Nom** : Nom du produit (ex: Ordinateur Portable)
   - **Prix Unitaire** : Prix de vente (ex: 50000)

3. **Remplir les Informations Optionnelles**
   - **Description** : Description détaillée
   - **Catégorie** : Catégorie du produit (ex: Électronique)
   - **Seuil Stock Minimum** : Quantité d'alerte (ex: 5)

4. **Enregistrer**
   - Cliquez sur **Enregistrer**
   - Un message de confirmation s'affiche
   - Le produit apparaît dans la liste
