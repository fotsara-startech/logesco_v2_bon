========================================
  LOGESCO - REINITIALISATION BASE
  Guide Rapide
========================================

PROBLEME RESOLU
===============
Le package client contient maintenant une base 100% vierge.
Si jamais des donnees de test sont presentes, vous pouvez
reinitialiser facilement.

SCRIPTS DISPONIBLES
===================

1. VERIFICATION
   ------------
   verifier-base-vierge.bat
   
   Verifie si la base est vierge ou contient des donnees.
   
   Resultat attendu:
   - 1 utilisateur (admin)
   - 0 produits
   - 0 ventes
   - 0 clients
   - 0 fournisseurs
   - 1 caisse

2. REINITIALISATION (Version Developpement)
   -----------------------------------------
   reinitialiser-base-donnees.bat
   
   Utilisation: Depuis la racine du projet
   
   Ce script va:
   - Sauvegarder l'ancienne base automatiquement
   - Supprimer toutes les donnees
   - Creer une nouvelle base vierge
   - Verifier que tout est correct

3. REINITIALISATION (Version Client)
   ----------------------------------
   REINITIALISER-BASE-DONNEES.bat
   (Inclus dans le package client)
   
   Utilisation: Depuis le dossier LOGESCO-Client-Optimise
   
   Meme fonctionnement que la version dev, mais avec:
   - Double confirmation (securite renforcee)
   - Messages adaptes pour le client
   - Verification emplacement automatique

UTILISATION RAPIDE
==================

Etape 1: Verifier
-----------------
verifier-base-vierge.bat

Si la base contient des donnees de test, passez a l'etape 2.

Etape 2: Reinitialiser
----------------------
reinitialiser-base-donnees.bat
(ou REINITIALISER-BASE-DONNEES.bat dans le package client)

Etape 3: Confirmer
------------------
Version Dev: Tapez OUI
Version Client: Tapez REINITIALISER puis OUI

Etape 4: Attendre
-----------------
Le script fait tout automatiquement:
[1/7] Verification prerequis
[2/7] Verification emplacement
[3/7] Arret de LOGESCO
[4/7] Sauvegarde ancienne base
[5/7] Suppression ancienne base
[6/7] Creation nouvelle base VIERGE
[7/7] Verification nouvelle base

Etape 5: Demarrer
-----------------
DEMARRER-LOGESCO.bat

Etape 6: Personnaliser
----------------------
1. Connexion: admin / admin123
2. Parametres > Entreprise (personnaliser)
3. Parametres > Utilisateurs > Changer mot de passe admin
4. Creer vos utilisateurs
5. Ajouter vos produits

SAUVEGARDE AUTOMATIQUE
======================

Emplacement: backend\database\backups\
Format: logesco_backup_YYYYMMDD_HHMMSS.db

Exemple: logesco_backup_20260302_143025.db

Pour restaurer:
1. Arreter LOGESCO
2. Copier le fichier depuis backups\
3. Coller dans backend\database\
4. Renommer en logesco.db
5. Redemarrer LOGESCO
