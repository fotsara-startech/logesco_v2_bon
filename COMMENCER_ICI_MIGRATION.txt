═══════════════════════════════════════════════════════════════
  SECTION A: CREER LE PACKAGE (Développeur)
═══════════════════════════════════════════════════════════════

ETAPE 1: Créer le package
--------------------------

Double-clic sur:
📄 preparer-pour-client-CORRIGE.bat

Attendez 2-3 minutes.

Vous devez voir:
✅ Package CORRIGE (sans Prisma pre-genere, sans base vierge)


ETAPE 2: Vérifier le package
-----------------------------

Double-clic sur:
📄 VERIFIER-PACKAGE-CORRIGE.bat

Vous devez voir:
✅✅✅ PACKAGE CORRECT ✅✅✅

Si vous voyez des erreurs:
❌ Corrigez et recommencez l'étape 1


ETAPE 3: Distribuer le package
-------------------------------

Le package est dans:
📁 release\LOGESCO-Client-CORRIGE\

Copiez ce dossier complet:
- Sur une clé USB
- Ou envoyez-le au client
- Ou partagez-le par réseau

Incluez aussi:
📄 MIGRER-VERS-CORRIGE.bat
📄 GUIDE_MIGRATION_VISUEL.txt


═══════════════════════════════════════════════════════════════
  SECTION B: MIGRER UN CLIENT
═══════════════════════════════════════════════════════════════

ETAPE 1: Préparer
-----------------

Vous avez reçu:
📁 LOGESCO-Client-CORRIGE\
📄 MIGRER-VERS-CORRIGE.bat
📄 GUIDE_MIGRATION_VISUEL.txt (optionnel)

Allez dans le dossier CLIENT (celui avec VOS données):
📁 E:\Stage 2025\LOGESCO-Client-OPTIMISE\

Copiez-y:
📁 LOGESCO-Client-CORRIGE\
📄 MIGRER-VERS-CORRIGE.bat

Structure finale:
📁 E:\Stage 2025\LOGESCO-Client-OPTIMISE\
├── 📁 backend\              ← Ancien (VOS données)
├── 📁 app\                  ← Ancien
├── 📁 LOGESCO-Client-CORRIGE\  ← NOUVEAU
└── 📄 MIGRER-VERS-CORRIGE.bat  ← NOUVEAU


ETAPE 2: Migrer
---------------

Dans le dossier CLIENT, double-clic sur:
📄 MIGRER-VERS-CORRIGE.bat

Suivez les étapes affichées (7 étapes).

À la fin, vous devez voir:
✅ Produits: 165 (votre nombre)

Si vous voyez "Produits: 0":
❌ Consultez GUIDE_MIGRATION_VISUEL.txt section DEPANNAGE


ETAPE 3: Démarrer
-----------------

Double-clic sur:
📄 DEMARRER-LOGESCO.bat

Première fois: 15-20 secondes
Fois suivantes: 7-9 secondes

Connexion:
Utilisateur: admin
Mot de passe: admin123

Vérifiez que vos données s'affichent!

PROBLEME: "Package CORRIGE non trouvé"
---------------------------------------
→ Vérifiez que vous êtes dans le dossier CLIENT

PROBLEME: "Produits: 0" après migration
----------------------------------------
→ Exécutez: REGENERER-PRISMA-SIMPLE.bat
→ Redémarrez l'application


PROBLEME: Données visibles en ligne mais pas dans l'app
--------------------------------------------------------
→ Exécutez: REGENERER-PRISMA-SIMPLE.bat
