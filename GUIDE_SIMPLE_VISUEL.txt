Le problème:
------------
Le package "optimisé" contient un Prisma pré-généré pour
une base VIDE.

Même si vous copiez votre base avec 165 produits, Prisma
utilise l'ancien client qui ne voit rien.


La solution:
------------
Supprimer le Prisma pré-généré et le régénérer avec
VOTRE base de données.
