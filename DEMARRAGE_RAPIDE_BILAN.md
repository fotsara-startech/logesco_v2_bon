# 🚀 Démarrage Rapide - Bilan Comptable d'Activités

## ✅ Module prêt à utiliser !

Le module de **Bilan Comptable d'Activités** est maintenant entièrement implémenté et intégré dans LOGESCO v2.

## 📱 Comment accéder au module

### 1. Lancer l'application
```bash
cd logesco_v2
flutter run -d windows
```

### 2. Navigation dans l'interface
1. **Ouvrir le menu** : Cliquer sur l'icône hamburger (☰) en haut à gauche
2. **Aller aux rapports** : Chercher la section "RAPPORTS"
3. **Cliquer sur "Bilan Comptable"** : Icône 📊 avec texte "Bilan Comptable"

## 🎯 Utilisation du module

### Étape 1 : Sélection de la période
- **Périodes rapides** : Cliquer sur les boutons prédéfinis
  - Aujourd'hui / Hier
  - Cette semaine / Semaine dernière  
  - Ce mois / Mois dernier
  - Ce trimestre / Cette année

- **Période personnalisée** : 
  - Cliquer sur "Date de début" → Sélectionner dans le calendrier
  - Cliquer sur "Date de fin" → Sélectionner dans le calendrier

### Étape 2 : Génération du bilan
- Cliquer sur le bouton **"Générer le bilan"**
- Attendre le chargement (quelques secondes)
- Le bilan s'affiche automatiquement

### Étape 3 : Consultation du rapport
Le bilan contient 6 sections principales :

1. **📋 Résumé Exécutif**
   - Statut général (Excellent/Bon/Modéré/Attention/Critique)
   - Indicateurs clés avec tendances
   - Points saillants de la période

2. **🛒 Analyse des Ventes**
   - Chiffre d'affaires total
   - Nombre de ventes et vente moyenne
   - Ventes par catégorie
   - Top 5 des produits les plus vendus

3. **💰 Analyse des Bénéfices**
   - Marge brute et bénéfice net
   - Marge de profit en pourcentage
   - Répartition des coûts
   - Évolution par rapport à la période précédente

4. **💳 Mouvements Financiers**
   - Total des entrées et sorties
   - Flux de trésorerie net
   - Mouvements par catégorie

5. **👥 Dettes Clients**
   - Total des dettes en cours
   - Nombre de clients débiteurs
   - Top 5 des principaux débiteurs

6. **💡 Recommandations**
   - Actions suggérées pour améliorer les performances
   - Conseils personnalisés selon la situation

### Étape 4 : Export PDF (optionnel)
1. Cliquer sur le bouton **"Export PDF"** (bouton rouge flottant)
2. Attendre la génération (30 secondes max)
3. Choisir l'action :
   - **"Ouvrir"** : Ouvre le PDF avec l'application par défaut
   - **"Partager"** : Partage par email, WhatsApp, etc.

## 📄 Contenu du PDF généré

Le PDF contient **4 pages professionnelles** :

### Page 1 : Résumé Exécutif
- En-tête avec informations de l'entreprise
- Statut général avec code couleur
- Tableau des indicateurs clés
- Points saillants de la période

### Page 2 : Analyse des Ventes
- Métriques de ventes (graphiques)
- Tableau des ventes par catégorie
- Liste des produits les plus vendus

### Page 3 : Mouvements Financiers et Bénéfices
- Résumé des flux de trésorerie
- Mouvements par catégorie
- Analyse détaillée des bénéfices et tendances

### Page 4 : Dettes Clients et Recommandations
- Résumé des dettes clients
- Tableau des principaux débiteurs
- Liste des recommandations d'amélioration

## ⚡ Conseils d'utilisation

### ✅ Bonnes pratiques
- **Générer régulièrement** : Hebdomadaire pour le suivi, mensuel pour les rapports
- **Comparer les périodes** : Utiliser différentes périodes pour voir les tendances
- **Exporter en PDF** : Conserver les bilans importants pour l'historique
- **Suivre les recommandations** : Appliquer les conseils suggérés

### ⚠️ Points d'attention
- **Données en temps réel** : Le bilan reflète les données actuelles de la base
- **Période cohérente** : S'assurer que la période sélectionnée contient des données
- **Connexion requise** : Le module nécessite une connexion au backend

## 🔧 Dépannage rapide

### Problème : "Aucune donnée pour la période"
**Solution** : Vérifier qu'il y a des ventes ou mouvements sur la période sélectionnée

### Problème : "Erreur lors de la génération"
**Solution** : Vérifier la connexion internet et réessayer

### Problème : "Export PDF lent"
**Solution** : Patienter, la génération peut prendre jusqu'à 30 secondes

### Problème : "Impossible d'ouvrir le PDF"
**Solution** : Installer une application de lecture PDF ou utiliser "Partager"
