================================================================================
  PROCHAINES ETAPES
================================================================================

1. REDEMARRER LE BACKEND
   cd backend
   npm start

2. TESTER DANS L'APPLICATION FLUTTER
   - Se connecter en tant qu'admin
   - Menu → Paramètres de l'entreprise
   - Ajouter un slogan: "Votre satisfaction, notre priorité"
   - Cliquer sur "Sélectionner un logo"
   - Choisir une image
   - Sauvegarder

3. VERIFIER LA SAUVEGARDE
   cd backend
   node test-logo-slogan-save.js

================================================================================
  FONCTIONNALITES
================================================================================

SLOGAN:
- Champ texte multiligne (2 lignes)
- Optionnel
- Max 200 caractères
- Affiché en pied de page des factures

LOGO:
- Sélection d'image (PNG, JPG, etc.)
- Max 5MB
- Optionnel
- Affiché en en-tête des factures A4/A5

================================================================================
  PROBLEMES COURANTS
================================================================================

PROBLEME: Le slogan ne se sauvegarde pas
SOLUTION: 
  1. Vérifier que le backend est redémarré
  2. Vérifier les logs Flutter
  3. Consulter GUIDE_TEST_LOGO_SLOGAN_FLUTTER.md

PROBLEME: Le logo ne se sélectionne pas
SOLUTION:
  1. Vérifier que file_picker est installé (flutter pub get)
  2. Vérifier les permissions Windows
  3. Essayer avec une image plus petite
