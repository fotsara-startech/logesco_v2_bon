PROBLEMES COURANTS:
───────────────────

PROBLEME: "Invalid column name"
   → Noms de colonnes ne correspondent pas
   → Solution: npx prisma db pull --force

PROBLEME: "Table not found"
   → Schema Prisma incorrect
   → Solution: npx prisma db push --accept-data-loss

PROBLEME: "Prisma Client not generated"
   → Client Prisma manquant
   → Solution: npx prisma generate

PROBLEME: "Database locked"
   → Autre processus utilise la BD
   → Solution: taskkill /f /im node.exe
