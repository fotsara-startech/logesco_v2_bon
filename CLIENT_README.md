# LOGESCO v2 - Guide Utilisateur

## 🎯 Installation Simple

### Étape 1: Télécharger
Téléchargez le fichier `LOGESCO-v2-Setup.exe`

### Étape 2: Installer
1. Double-cliquez sur `LOGESCO-v2-Setup.exe`
2. Cliquez sur "Suivant"
3. Cliquez sur "Installer"
4. Attendez quelques secondes
5. Cliquez sur "Terminer"

### Étape 3: Utiliser
1. Lancez LOGESCO depuis votre bureau ou le menu Démarrer
2. L'application démarre automatiquement
3. Connectez-vous avec vos identifiants

**C'est tout! Aucune configuration supplémentaire nécessaire.**

## 🔐 Premier Démarrage

Au premier lancement, LOGESCO va:
- Initialiser automatiquement la base de données
- Créer un compte administrateur par défaut
- Préparer l'application pour utilisation

**Temps d'initialisation: ~10 secondes**

## 💡 Informations Importantes

### Aucune Connexion Internet Requise
LOGESCO fonctionne 100% en local sur votre ordinateur. Aucune connexion internet n'est nécessaire.

### Vos Données Sont Sécurisées
Toutes vos données sont stockées localement sur votre ordinateur. Rien n'est envoyé sur internet.

### Sauvegarde Automatique
Vos données sont automatiquement sauvegardées dans la base de données locale.

## 🆘 Besoin d'Aide?

### L'application ne démarre pas
1. Redémarrez votre ordinateur
2. Relancez LOGESCO
3. Si le problème persiste, contactez le support

### J'ai oublié mon mot de passe
Contactez votre administrateur système ou le support technique.

### Comment faire une sauvegarde?
Vos données sont dans: `C:\Users\[VotreNom]\AppData\Local\LOGESCO\backend\database\`
Copiez le fichier `logesco.db` pour faire une sauvegarde.
