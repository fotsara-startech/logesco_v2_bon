Si vous avez déjà distribué l'ancien package, créez ce script
dans le dossier du client:

REPARER-CLIENT.bat:
-------------------
@echo off
echo Reparation client...
cd backend
if exist "node_modules\.prisma" rmdir /s /q "node_modules\.prisma"
if exist "node_modules\@prisma\client" rmdir /s /q "node_modules\@prisma\client"
call npx prisma generate
echo ✅ Repare!
pause

Puis lancez DEMARRER-LOGESCO.bat normalement.
