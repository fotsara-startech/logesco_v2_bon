# 📊 Guide Utilisateur - Import/Export Excel des Produits

## 🎯 Accès à la fonctionnalité

1. **Ouvrir l'application LOGESCO**
2. **Aller dans "Gestion des Produits"**
3. **Cliquer sur le menu ⋮ (trois points)** en haut à droite
4. **Sélectionner "Import/Export Excel"**

## 📤 Exporter vos produits

### Étapes simples :
1. Dans la page Import/Export, cliquer sur **"Exporter tous les produits"**
2. Attendre la génération du fichier (quelques secondes)
3. Choisir **"Partager"** pour envoyer le fichier ou **"Fermer"** pour le sauvegarder

### Ce que vous obtenez :
- **Fichier Excel** avec tous vos produits
- **Colonnes complètes** : référence, nom, prix, catégorie, etc.
- **Format professionnel** avec en-têtes colorés
- **Prêt à partager** ou archiver

## 📥 Importer des produits

### Première fois - Utiliser le template :
1. Cliquer sur **"Template"** pour télécharger le modèle
2. **Ouvrir le fichier** dans Excel ou LibreOffice
3. **Remplacer les exemples** par vos vrais produits
4. **Sauvegarder** le fichier

### Format requis :
| Colonne | Obligatoire | Exemple |
|---------|-------------|---------|
| Référence | ✅ Oui | REF001 |
| Nom | ✅ Oui | Mon Produit |
| Prix Unitaire | ✅ Oui | 25.99 |
| Description | ❌ Non | Description détaillée |
| Prix Achat | ❌ Non | 15.50 |
| Catégorie | ❌ Non | Électronique |
| Est Actif | ❌ Non | Oui |
| Est Service | ❌ Non | Non |

### Importer le fichier :
1. Cliquer sur **"Importer depuis Excel"**
2. **Sélectionner votre fichier** (.xlsx ou .xls)
3. **Vérifier l'aperçu** des produits détectés
4. **Supprimer** les lignes incorrectes si nécessaire
5. Cliquer sur **"Confirmer l'import"**

## ⚠️ Points importants

### ✅ Bonnes pratiques :
- **Références uniques** : Chaque produit doit avoir une référence différente
- **Prix valides** : Utiliser des nombres (ex: 25.99, pas 25,99€)
- **Oui/Non** : Pour Est Actif et Est Service, écrire "Oui" ou "Non"
- **Vérifier l'aperçu** avant de confirmer l'import

### ❌ Erreurs courantes :
- **Référence déjà existante** → Le produit sera ignoré
- **Prix invalide** → La ligne sera ignorée
- **Champs obligatoires vides** → La ligne sera ignorée

## 🔧 Dépannage

### "Aucun produit valide trouvé"
- Vérifier que les colonnes ont les bons noms
- S'assurer que les champs obligatoires sont remplis
- Utiliser le template fourni

### "Référence déjà existante"
- Changer la référence pour qu'elle soit unique
- Ou supprimer la ligne de l'aperçu avant import

### "Erreur de format"
- Vérifier que les prix sont des nombres
- Utiliser le point (.) pour les décimales, pas la virgule (,)

## 💡 Conseils d'utilisation

### Pour migrer depuis un autre système :
1. **Exporter** vos données depuis l'ancien système
2. **Adapter** le format au template LOGESCO
3. **Tester** avec quelques produits d'abord
4. **Importer** le fichier complet

### Pour faire une sauvegarde :
1. **Exporter** régulièrement vos produits
2. **Sauvegarder** les fichiers Excel
3. **Partager** avec votre équipe si nécessaire

### Pour des mises à jour en masse :
1. **Exporter** les produits actuels
2. **Modifier** les données dans Excel
3. **Supprimer** les anciens produits si nécessaire
4. **Importer** les données mises à jour
