## Structure des fichiers de traduction

```
logesco_v2/lib/core/translations/
├── app_translations.dart       # Classe principale
├── fr_translations.dart        # Traductions françaises
└── en_translations.dart        # Traductions anglaises
```

## Utilisation dans le code

### Traduction simple
```dart
Text('procurement_title'.tr)
```

### Traduction avec paramètre
```dart
Text('procurement_order_details'.tr.replaceAll('@number', commande.numeroCommande))
```

### Traduction conditionnelle
```dart
Text(isComplete ? 'procurement_complete'.tr : 'procurement_in_progress'.tr)
```

## Maintenance future

### Ajout d'une nouvelle langue
1. Créer un nouveau fichier (ex: `es_translations.dart`)
2. Copier toutes les clés depuis `fr_translations.dart`
3. Traduire les valeurs en espagnol
4. Ajouter la locale dans `app_translations.dart`

### Ajout d'une nouvelle clé
1. Ajouter la clé dans `fr_translations.dart`
2. Ajouter la traduction dans `en_translations.dart`
3. Utiliser `.tr` dans le code

### Modification d'une traduction
1. Modifier la valeur dans le fichier de traduction
2. Aucune modification de code nécessaire
3. Hot reload pour voir les changements
