   → Crée le package avec base propre

-------------------------------
Inspectez la base créée :

   cd release\LOGESCO-Client-Optimise\backend

------------------------------
Testez le package :

   cd release\LOGESCO-Client-Optimise
   DEMARRER-LOGESCO.bat

   → Connectez-vous avec admin/admin123
   → Vérifiez que l'application fonctionne
   → Confirmez que la base est vide


ÉTAPE 6 : DISTRIBUER AU CLIENT
-------------------------------
Une fois satisfait :

   → Distribuez le package au client
   → Avec confiance !
